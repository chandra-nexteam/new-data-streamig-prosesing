# Final Project

Data stream processing project demonstrating real-time data handling and analysis from various sources (MySQL, PostgreSQL, MongoDB)

### Tech Stack:

- Python
- PostgreSQL
- MySQL
- MongoDB
- Kafka
- Spark
- GCP (Google Sheets API, Google Drive API)
- Looker
